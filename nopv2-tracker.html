REST API Documentation

Base URL: https://api.nopv2tracker.org/v1

GET/responses
List all outbreak responses with optional filters

Query Parameters

ParameterTypeDescription
countrystringFilter by country name (e.g. "Angola")
statusstringFilter by campaign status: completed, planned
roundstringFilter by round: R1, R2, R3, Mop-up
regionstringFilter by WHO region (e.g. "AFRO")
searchstringFull-text search across SIA codes and admin regions
pageintegerPage number (default: 1)
per_pageintegerResults per page (default: 50, max: 200)

Example Request

curl "https://api.nopv2tracker.org/v1/responses?country=Angola&status=completed"

Example Response

{
  "data": [
    {
      "id": 1,
      "sia_code": "AGO-2024-001",
      "country": "Angola",

}

POST/responses
Create a new outbreak response record

Request Body (JSON)

{
  "sia_code": "AGO-2025-003",
  "country": "Angola",
  "region": "AFRO",
  "vaccine_type": "nOPV2",
  "round_number": "R1",
  "campaign_status": "Planned - no date change",
  "planned_date": "2025-09-01",
  "start_date": "2025-09-01",
  "end_date": "2025-09-03",
  "target_pop_polis": 500000,
  "admin_regions": ["LUANDA", "BENGO"]
}

Response (201 Created)

{
  "data": { "id": 10, "sia_code": "AGO-2025-003", ... },
  "message": "Response created successfully"
}

POST/upload
Upload a revised Excel spreadsheet

Request

curl -X POST "https://api.nopv2tracker.org/v1/upload" \
  -H "Authorization: Bearer <token>" \
  -F "file=@nopv2_response_updated.xlsx" \
  -F "mode=merge"
FieldTypeDescription
filefileThe .xlsx file to upload
modestringreplace (overwrite all) or merge (upsert by SIA code)

Response (200 OK)

{
  "message": "Upload processed successfully",
  "stats": {
    "rows_parsed": 12,
    "rows_created": 3,
    "rows_updated": 9,
    "errors": []
  }
}
DELETE/responses/{id}
Delete a response record

Example Request

curl -X DELETE "https://api.nopv2tracker.org/v1/responses/10" \
  -H "Authorization: Bearer <token>"

Response (200 OK)

{
  "message": "Response deleted successfully"
}
